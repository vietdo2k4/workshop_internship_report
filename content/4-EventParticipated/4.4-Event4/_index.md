# Event Notes: FCAJ Community Day 27/06

### Session Goals

- Exploring agent-driven operations (AgenticOps) and automated AI Agents on cloud platforms.
- Learning Voice Agent deployment architectures and developer assistant tools for DevOps.
- Reviewing security strategies for large language models using Private Model Context Protocol (MCP).
- Analyzing productivity models enabled by generative AI to improve enterprise efficiency.

### Speaker List

- **Steve Tran** - AgenticOps for your Cloud
- **Trung Vu, Nghi Danh, Kiet Tran** - Building Voice Agent at Scale
- **Bao Phan, Nguyen Nguyen** - AWS DevOps Agent
- **Truong (Wayne) Tran, Anh Dang (Akie)** - AI-Powered Productivity
- **Nguyen Duc Toan, Danh Hoang Huu Nghi** - Building secure private MCP for Quick

### Featured Content

#### AgenticOps for your Cloud

- Leveraging autonomous AI Agents to manage cloud resources, resolve service incidents, and minimize manual supervision.
- Agents can proactively detect anomalies and trigger appropriate remediation actions without waiting for an engineer to step in.

#### Building Voice Agent at Scale

- Engineering real-time conversational agents designed to scale gracefully with spike audio loads.
- Covers the core pipeline components: Speech-to-Text, an LLM for context handling, Text-to-Speech, and how to wire them together seamlessly.

#### AWS DevOps Agent

- Utilizing AWS DevOps Agents to aid system administrators in monitoring resources and responding to alerts.
- Integrates with CloudWatch and other AWS services to enable automated monitoring and fast event response.

#### AI-Powered Productivity

- Applying AI models for workforce planning, optimizing work distribution, and aiding data-driven decision processes.
- Pairs real enterprise data with AI to shift decisions from gut feeling to evidence-based reasoning.

#### Building secure private MCP for Quick

- Crafting secure private MCP bridges to safeguard local business data during interaction with cloud-based AI.
- Addresses the real concern of sensitive corporate data being exposed when connecting to third-party LLMs in the cloud.

### Key Takeaways

- **Mindset & Personal Skills**: Embracing an agent-first paradigm for automation; advocating for data protection and security-first engineering in AI solutions.
- **Architecture & Engineering**: Learning the architectural mechanics of agent runtimes, real-time voice ingestion pipelines, and private MCP configuration patterns.

### Action Plan & Future Direction

- Investigating DevOps Agent designs to automate basic checking tasks on personal servers.
- Exploring MCP standards to connect local context libraries with commercial LLMs securely.
